# 五子棋Alpha-Beta算法演示网站

一个功能完整的五子棋游戏，集成了先进的Alpha-Beta剪枝算法，提供直观的算法演示和参数调整功能。

## 🎮 功能特色

### 核心功能
- **智能对战**: 基于Alpha-Beta剪枝算法的高水平AI对手
- **实时演示**: 可视化展示AI搜索过程和决策逻辑
- **参数可调**: 用户可以调整算法参数，观察不同设置下的AI表现
- **详细日志**: 记录算法运行的详细信息，包括搜索节点、剪枝次数等

### 技术特点
- **Alpha-Beta剪枝**: 优化的Minimax算法，大幅提高搜索效率
- **启发式评估**: 基于棋型识别的智能评估函数
- **候选点筛选**: 只评估有潜力的位置，减少计算量
- **可配置权重**: 活四、活三、眠三等棋型的权重可调

## 🎯 使用指南

### 游戏界面
1. **开始游戏**: 点击"新局"按钮开始新的对局
2. **玩家落子**: 点击棋盘上的空位进行落子
3. **AI对战**: 点击"AI搜索"让AI进行思考和落子
4. **悔棋功能**: 点击"后退"可以撤销上一步落子

### 参数设置
在设置页面中可以调整以下参数：

#### 搜索参数
- **搜索深度**: 控制AI思考的层数（1-10层）
- **候选点数**: 每次搜索考虑的候选位置数量
- **搜索范围**: 只评估距离已有棋子多远的空位

#### 启发函数权重
- **活五**: 五子连珠的权重（默认100000）
- **活四**: 两端开放的连续四子权重（默认5000）
- **冲四**: 一端被阻挡的四子权重（默认400）
- **活三**: 两端开放的连续三子权重（默认400）
- **眠三**: 一端被阻挡的三子权重（默认20）
- **活二**: 两端开放的连续二子权重（默认10）
- **眠二**: 一端被阻挡的二子权重（默认5）
- **对手威胁系数**: 评估对手威胁的权重系数（默认-0.8）

### 日志分析
日志页面提供详细的算法运行信息：
- **搜索节点数**: 算法评估的节点总数
- **剪枝次数**: Alpha-Beta剪枝节省的计算量
- **搜索时间**: 每次AI思考的耗时
- **详细日志**: 每一步的评估分数和决策过程

## 🧠 算法原理

### Alpha-Beta剪枝
Alpha-Beta剪枝是对Minimax算法的优化，通过剪掉不可能影响最终决策的分支来减少搜索量：

```
初始: α = -∞, β = +∞
搜索过程中:
- Max节点更新α值
- Min节点更新β值
- 当α ≥ β时进行剪枝
```

### 启发式评估
评估函数基于棋型识别，为不同的棋型分配权重：

```javascript
总分 = 己方棋型得分 - 对手棋型得分 × 威胁系数

棋型权重:
- 活五: 100000 (必胜)
- 活四: 5000 (必胜威胁)
- 冲四: 400 (强威胁)
- 活三: 400 (潜在威胁)
- 眠三: 20 (弱威胁)
- 活二: 10 (发展潜力)
- 眠二: 5 (微弱潜力)
```

### 候选点生成
为了提高效率，AI不会评估棋盘上的所有空位：
1. 只考虑距离已有棋子一定范围内的位置
2. 对每个位置进行快速评估
3. 选择评分最高的N个位置进行深入搜索
